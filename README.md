## Getting Started まずはここから

### Dependencies 必要なもの

- Node
- npm

ここからインストールできます．  
日本語サイト https://nodejs.org/ja/download/

### Installing インストール

```sh
$ npm i -g stlap
```

インストールされていれば以下のコマンドでバージョンが表示されます．

```sh
$ stlap --version
```

### 使用例

テキストファイル

```
$ stlap story.txt
```
